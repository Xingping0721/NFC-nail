# overview
NFCを埋め込んだネイルアートを作る。そのNFCにスマホをかざすと、簡易的な名刺が表示されるようなシステムを作る

# motivation
- 名刺を忘れてしまい、相手方にも失礼なことをしてしまうことがあった。そこで、絶対に忘れないようにNFCを物理的に肌身離さずつけておけばいいことが分かったので、ネイルに埋め込む。
- また、実際に名刺交換したときの印象が大きいので、その効果も狙う

# やること
1. 5mm ×5mm くらいの小さ目のNFCを購入する。大体150円/個くらい
2. ネイルを塗る ->
   - 手順としては
     1. NFC
     2. ベースコート
     3. 色塗り
     4. トップコート
  初心者なので何個かNFCを壊してしまう可能性があるので、余分にNFCを買う
3. NFC ToolsというアプリでNFCに書き出し
4. テスト
